# О проекте

Эта вика автоматически собирает актуальные темы обсуждения после выхода
новых серий аниме и публикует их в удобном виде.

## Как это устроено

- **Движок вики:** [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) —
  статический сайт на Markdown.
- **Поиск трендов:** [Google Gemini API](https://ai.google.dev/) с инструментом
  **Google Search** (grounding) — модель ищет свежие обсуждения в сети и
  структурирует их.
- **Автоматизация:** GitHub Actions запускается по расписанию (cron) каждый день,
  проверяет, какие серии вышли, и обновляет соответствующие страницы.
- **Публикация:** обновлённый сайт автоматически деплоится на GitHub Pages.

## Как добавить новое аниме

1. Открой [`schedule.yaml`](https://github.com/USERNAME/anime-wiki) в корне репозитория.
2. Скопируй шаблон тайтла (он там закомментирован) и заполни поля:
   `slug`, `title`, `weekday`, `start_date`, `episodes` и т.д.
3. Закоммить изменения. При следующем запуске вики сама создаст страницу тайтла.

## Приватность

- Вики публична только если репозиторий публичный (нужно для бесплатного GitHub Pages).
- Хочешь закрыть от посторонних? Можно публиковать сайт приватно или отдать
  подруге собранные Markdown-файлы напрямую.
