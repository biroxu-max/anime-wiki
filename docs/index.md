## 🔍 Как это работает

1. Каждый день **GitHub Actions** запускает генератор по расписанию.
2. Скрипт ищет свежие обсуждения через **Tavily** (англоязычные форумы + японский 5ch)
   и структурирует их через **GLM** — реакции, теории, повороты сюжета, мемы, споры.
3. Страница коммитится в репозиторий и публикуется на GitHub Pages.

Подробнее — на странице [О проекте](about.md).
