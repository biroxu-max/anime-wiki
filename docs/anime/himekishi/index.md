# Himekishi wa Barbaroi no Yome

**Принцесса-рыцарь — невеста варвара** · 姫騎士は蛮族の嫁

!!! abstract "О чём"
    «Просто убей меня!» — эти слова произносит Серафина де Лавийян, сильнейшая женщина-рыцарь Запада, когда её берёт в плен предводитель варваров. Вместо казни он предлагает ей стать его невестой.
